# javascript之闭包

## 闭包的概念

```
闭包（closure）是 JavaScript 的一种语法特性。

关于闭包，有一种经典的提法——“闭包是代码块和创建该代码块的上下文（环境）中数据的结合”。

闭包就是在函数内部定义函数，内部的函数可访问其外部函数的作用域。下面是在程序中实现闭包的例子。
```

> ```js
> function outer(name){ // 外部的函数
>  var msg="hello";
>  function inner(){ // 内部函数
>    alert(msg+" "+name);
>   }
>  return inner(); // 返回内部函数
> }
> var clos=outer("WANGERN");
> clos();
> ```
